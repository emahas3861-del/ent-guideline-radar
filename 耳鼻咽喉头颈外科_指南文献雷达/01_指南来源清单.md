# 指南来源清单

## 国际综合与肿瘤

| 来源 | 重点 | 备注 |
|---|---|---|
| NCCN Guidelines | 头颈癌、甲状腺癌、支持治疗 | 常更新；部分内容需账号或机构权限 |
| ESMO Clinical Practice Guidelines | 头颈癌、甲状腺癌、罕见肿瘤 | 适合提炼治疗路径变化 |
| ASCO Guidelines | 肿瘤系统治疗、随访、支持治疗 | 关注免疫/靶向治疗 |
| WHO Classification / IARC | 肿瘤病理分类 | 影响诊断命名和风险分层 |
| AJCC / UICC TNM | 肿瘤分期 | 关注新版分期变化 |

## 耳鼻咽喉头颈外科专业组织

| 来源 | 重点 | 备注 |
|---|---|---|
| AAO-HNSF | 美国耳鼻咽喉头颈外科临床实践指南 | 中耳炎、鼻窦炎、扁桃体、突聋等 |
| AHNS | 头颈肿瘤、外科质量与随访 | 适合头颈肿瘤专题 |
| EPOS / European Position Paper | 鼻窦炎、鼻息肉 | 鼻科重点来源 |
| EAONO | 耳科、神经耳科、胆脂瘤 | 耳科重点来源 |
| ICAR | 鼻炎、鼻窦炎、嗅觉、过敏 | 多为共识/综述型文件 |
| ERS / ATS | 睡眠、气道、呼吸相关指南 | OSA 和气道问题可补充 |

## 甲状腺与内分泌相关

| 来源 | 重点 | 备注 |
|---|---|---|
| ATA | 甲状腺结节、分化型甲状腺癌、妊娠甲状腺疾病 | 头颈外科常用 |
| ETA | 甲状腺结节、甲状腺癌、分子检测 | 欧洲路径补充 |
| ACR TI-RADS | 甲状腺超声风险分层 | 与穿刺决策相关 |
| Bethesda System | 甲状腺细胞学 | 影响 Bethesda III/IV/V 管理 |

## 国内来源

| 来源 | 重点 | 备注 |
|---|---|---|
| 中华医学会耳鼻咽喉头颈外科学分会 | 国内指南/共识 | 需核对期刊原文 |
| 中国抗癌协会 / CSCO | 头颈肿瘤、甲状腺癌 | 适合国内临床路径 |
| 中华耳鼻咽喉头颈外科杂志 | 国内指南、专家共识 | 重点手动检索 |
| 中国知网 / 万方 / 医脉通指南库 | 中文指南与共识 | 需人工核验原文来源 |

## 文献数据库

| 来源 | 用途 | 优先级 |
|---|---|---|
| PubMed | 生物医学文献、指南、系统综述 | 第一优先 |
| Cochrane Library | 系统综述 | 高优先 |
| CrossRef | DOI 与出版信息补充 | 中 |
| ClinicalTrials.gov | 新药/新技术临床试验 | 专题使用 |
| medRxiv/bioRxiv | 预印本 | 只作线索，不作正式依据 |
